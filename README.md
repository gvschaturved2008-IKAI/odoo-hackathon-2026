# AssetFlow

AssetFlow is a smart asset management system built using Flask and SQLite.

## Features

- Asset registration and tracking
- Employee management
- Resource booking with overlap prevention
- Asset allocation
- Maintenance request management
- Audit cycle tracking
- User roles (Admin, Manager, Employee)
- Notifications and activity feed
- Dashboard and reports

## Technologies Used

- Python
- Flask
- SQLite
- HTML/CSS

## How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open the browser and visit:
   http://127.0.0.1:5000
